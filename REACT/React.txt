RULES FOR jsx;
    1.One Root element.
            --Must return a Single Root Tag.
            return(
                <div>
                <h1>qwertyui<h1/>
                </div>
            )

    2.Use className Instead of class
            --Because class is a reserved word in js.
        

    3.Attributes are in CamelCase
    ex:onClick,tabIndex,

    4.Expressions inside  { }
    ex:
    const name="Elango";
    return <h1>Hello,{name}<h1/>;

    5.jsx must be Closed
    ex:
    <img src={"logo.png"}/>

    6.Conditional Rendering
    (using && or ? :)

HOOKS:
    1.useState - For State Management
        Lets you add state (variables that react watches and re-renders when changed)

    2.useEffect 
            -useEffect(()=>{},[])
    
    - Runs on Every Render
    useEffect(()=>{
        console.log("Component updated")
    });

    --Runs only on First Mount
        useEffect(()=>{},[])
    
    Run when a Depedencies changes
        -useEffect(()=>{},[count])
